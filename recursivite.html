Fonction itérative

  Itérer, c'est répéter n fois un processus en faisant changer la valeur des variables jusqu'à obtention du résultat.

Un calcul itératif se programme par une boucle (for ou while...).

Exemple de fonction itératif pour le calcul de factorielle en C

int fact(n) {
	int i = 0;
	int result = 1;
	while (i < n){
		i = i + 1;
		result = result ∗ i;
	}
	return(result); 
}

Fonction récursive

  Tout d'abord, toute fonction récursive possède un équivalent itératif. Cependant, elle est plus lente que celle-ci.

Ensuite, une fonction est dite récursive si elle s'appelle elle-même.

 Récursivité non terminale

    Une fonction récursive dite non terminale, si le résultat de l'appel récursif est utilisé pour réaliser un traitement. En effet, on utilise la plupart du temps les valeurs précédentes de n pour calculer celle de n.

Exemple de fonction récursive non terminale pour le calcul de factorielle en C

int Factorielle(int n){
	int Z;
	if (n<1){
		return 1;
	}else{
		Z = Factorielle(n-1);
		return n*Z;
	}
}

 Récursivité terminale

   Une fonction récursive dite terminale, si aucun traitement n'est effectué à la remonté d'un appel récursif. En effet, pour cette fonction, il n'y a pas de besoin des valeurs précédentes de n.

Exemple de fonction récursive terminale pour le calcul de factorielle en C

int Factorielle(int n, int acc){
	if (n<1){
		return acc;
	}else{
		acc = acc*n;
		return Factorielle(n-1, acc);
	}
}
